      Updated account object.

      • address: string | Record<string, string[]>

        Account addresses. It can be a single address or a map of addresses per chain.

        If the address is a string, it's assumed to be under the 'eip155' namespace. Otherwise, it must be a map of addresses per chain, where the key is the chain ID (CAIP-2) and the value is an array of addresses.

        Example

        address: {
          // Different addresses per chain.
          'eip155:1': ['0x1234...'],
          'eip155:137': ['0x5678...'],
        }

        Example

        address: {
          // The address is the same across all 'eip155' chains.
          'eip155': ['0x1234...'],
        }

        Example

        // Assumed to be under the 'eip155' namespace.
        address: '0x1234...',
      • id: string
